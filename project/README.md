## Правила игры
- Гоблин появляется в случайной ячейке на 1 секунду
- Кликни по гоблину, чтобы получить +1 очко
- Пропустишь 5 появлений — игра окончена
- Курсор в виде молотка 🎯

## Запуск локально
```bash
yarn install
yarn start
